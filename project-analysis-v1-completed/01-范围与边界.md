## 1. V1 已完成范围

V1 验收包含以下**接入面与管理面**（侧栏无「（实验中）」标注）：

| 类别 | 内容 |
|------|------|
| Web 管理后台 | 工作台、知识库、『查询型』配置、**连接器管理**、Prompt 管理、会话管理、业务调试、**工具管理**、**能力路由**、嵌入式 Copilot 管理、OpenAPI 管理、**规则**、**审批中心**、**任务中心**、**审计中心**、用户与权限、租户管理、系统设置 |
| 嵌入式 Copilot | 配置、嵌入预览、实际嵌入页 `/copilot` |
| OpenAPI | 对外 `/openapi/v1/*` + 管理端应用接入与文档 |
| 运行时链路 | Agent Runtime、四类业务能力 Handler（经会话调试 / Copilot / OpenAPI 触达） |
| 治理 | JWT + RBAC、租户隔离、平台 LLM 接入、Prompt SSOT（基础设施） |

### 1.1 已通过正式入口验证的业务能力

| 能力 | 正式配置入口 | 正式运行入口 |
|------|--------------|--------------|
| 问答型 | 知识库 + 系统设置/模型接入 | 知识库/问答测试、会话调试、Copilot、OpenAPI |
| 查询型 | 『查询型』配置（连接器→ER→工具→查询测试） | 同上 + 业务调试 |
| 操作型 | **连接器管理** + **工具管理** + **能力路由**（规则/toolKind） | 会话调试、Copilot、OpenAPI、**业务调试/能力演示** |
| 流程型 | 工具管理（workflow）+ **任务中心** | 同上（需 Workflow Tool 已配置） |

操作型 **HTTP 类 Tool** 与 **能力路由** 已产品化；流程型任务编排与任务管理 UI 已产品化。Runtime Handler 已实现，验收通过 Copilot / 会话调试 / **能力演示** / OpenAPI 路径完成。

### 1.2 管理后台已产品化模块（原标注实验中，现已验收）

以下模块在初始方案或早期 V1 文档中曾为「实验中」或分散在其他菜单，**现均已移入已完成范围**：

| 初始章节 | 菜单 | 路由 | 模块文档 |
|----------|------|------|----------|
| §1.4 能力路由 | 能力路由 | `/routing/rules`、`/routing/test`、`/routing/capabilities` | [13-能力路由.md](./modules/13-能力路由.md) |
| §1.5 工具管理 | 工具管理 | `/tools/http-query`、`/tools` | [12-工具管理.md](./modules/12-工具管理.md) |
| §1.6 连接器管理 | 连接器管理 | `/connectors`（HTTP/通知）；只读库见『查询型』`/query/db-connectors` | [14-连接器管理.md](./modules/14-连接器管理.md) |
| §1.9 审计中心 | 审计中心 | `/audit`、… | [15-审计中心.md](./modules/15-审计中心.md) |
| §1.7 规则 | 规则 | `/rules`、`/rule-hits` | [16-规则与Policy.md](./modules/16-规则与Policy.md) |
| §1.8 审批中心 | 审批中心 | `/approvals`、`/approvals/history` | [17-审批中心.md](./modules/17-审批中心.md) |
| §1.3 任务中心 | 任务中心 | `/tasks`、`/tasks/tracking`、`/tasks/logs` | [18-任务中心.md](./modules/18-任务中心.md) |

**要点摘要**：

- **规则**：Policy 显式规则；命中留痕含 Copilot 路由级确认；侧栏在 OpenAPI 与审批中心之间
- **审批中心**：待确认 / 审批记录；Copilot 内联确认与管理端共用运行时；侧栏在规则与任务中心之间
- **任务中心**：任务列表 / 长任务跟踪 / 执行日志；流程型 Runtime 写入 task；侧栏在审批中心与审计中心之间
- **能力路由**：两阶段路由；`routing_rule.needConfirmation` 与 Policy 合并；规则 AI / 条件测试 / LLM 优化
- **工具管理**：`http_query` 独立菜单；AI 润色、导出对接文档、预制模板；统一 HTTP Invoker
- **连接器管理**：HTTP / 通知类独立菜单；与只读库连接器分入口维护
- **审计中心**：侧栏位于任务中心与用户与权限之间；四页只读查询；风险动作为审批 + Tool 调用等数据聚合

## 3. 相对初始方案的实现差异（全局）

| 主题 | 初始方案 | V1 现网 |
|------|----------|---------|
| 前端工程 | Next.js | **Vite + React + React Router**（`shellder-web-console`） |
| 知识库服务名 | `pathy-knowledge-server` | **wiki 知识库服务**（配置键 `knowledge.wikiBaseUrl`，见 `docs/knowledge-wiki-integration.md`） |
| 知识库菜单 | 「知识库管理与规则」合一 | **知识库** 与 **规则** 拆为独立顶级菜单（均已产品化） |
| 查询型配置 | 合并在连接器/工具管理 | 独立顶级菜单 **『查询型』配置**，复用 connector/tool 页面与 API |
| SQL 表白名单 | 表白名单 + 字段白名单 | 现网为 **表/字段黑名单** + ER 数据范围列映射 |
| 业务调试 | 无独立菜单 | 新增 **能力演示**（与 Copilot 嵌入契约对齐） |
| Prompt 子菜单 | 列表/详情/对比/绑定等多页 | 合并为 **模板列表 + 详情页**（版本/试跑/绑定在详情内） |
| OpenAPI 调用日志 | 独立菜单 | **应用详情页**「调用日志」卡片为主；侧栏仍保留 `/openapi/logs` 全局检索入口 |
| 租户详情/编辑/隔离 | 独立菜单项 | 路由存在（`/tenants/:id/*`），侧栏仅列列表与新建 |
| 连接器管理菜单 | 与工具/查询型合一或实验中 | **已产品化**；双入口 `/connectors` + `/query/db-connectors` |
| 能力路由 / 工具管理 | 早期实验中或分散 | **已产品化**；见 §1.2 |
| http_query 配置辅助 | 无 | **AI 润色**、**导出对接文档**、预制模板 |
| 路由规则调试 | 仅全链路 `/routing/test` | **conditions 即时测试** + **LLM 优化未命中条件** |
