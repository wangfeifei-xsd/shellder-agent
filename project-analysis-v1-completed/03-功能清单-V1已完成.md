## 1. Web 管理后台

### 1.1 工作台

**菜单**：工作台 / 首页（`/`）

**功能**：

- 展示工具成功率、失败率、平均响应时长（聚合 `tool_call_audit`）
- 展示高风险动作待确认列表（跳转 `/approvals`，该页为实验中）
- 展示最近异常任务（跳转 `/tasks`，实验中）
- 快捷入口：会话调试、工具管理、连接器管理、审计中心、技能书管理（后四项为实验中路由，受 RBAC 过滤）

**API**：`GET /api/v1/dashboard/summary?tenantId=`

---

### 1.2 会话管理

**菜单**：

- 会话管理 / 会话列表（`/sessions`）
- 会话管理 / 消息记录（`/sessions/messages`）
- 会话管理 / 调试台（`/sessions/debug`）
- （路由）会话详情（`/sessions/:id`）

**功能**（与初始 §1.2 对齐，已验收部分）：

| 子页 | 功能 |
|------|------|
| 会话列表 | 按租户、用户、状态、能力类型、时间筛选；标题、发起人、最近消息时间、状态 |
| 会话详情 | 完整消息流；模型回复、工具调用、确认节点、最终结果；关联任务（依赖实验中任务中心） |
| 消息记录 | 用户/系统/工具/确认消息；按类型筛选；时间线 |
| 调试台 | 发起调试会话；指定租户/用户/场景；SSE 流式；完整 Runtime 链路 |

**API 前缀**：`/api/v1/sessions`、`/api/v1/messages`；SSE `GET /api/v1/sessions/:id/stream`

---

### 1.2A 业务调试（V1 新增，初始清单无独立菜单）

**菜单**：业务调试 / 能力演示（`/capabilities`）

**功能**：

- 管理端换 Copilot Token，以与嵌入 Copilot **相同契约**演示四类能力
- 支持问答、查询等已配置能力的端到端验证

**API**：`POST /api/v1/capabilities/demo/copilot-token` → `/copilot/v1/*`

### 1.3 『查询型』配置（V1 菜单重组）

初始方案分散在 §1.5 工具管理、§1.6 连接器管理；V1 合并为独立顶级菜单：

| 子菜单 | 路由 | 对应初始章节 |
|--------|------|--------------|
| 数据库连接器 | `/query/db-connectors` | §1.6 只读库连接器 |
| 库表 ER 图 | `/query/db-er` | 查询型详细方案 §4 |
| 数据库连接工具 | `/query/db-channel-tools` | §1.5 Query Tool |
| 查询测试 | `/query/query-test` | §1.5 调用测试 + SQL 查询 |

**功能要点**：

- 只读库连接器 CRUD、**SELECT 1** 连通性测试
- information_schema 抽取、LLM 生成 ER 草稿、可视化编辑、发布
- 数据范围/限制字段配置（ER 列映射）
- Query Tool 绑定连接器；NL2SQL 预览、SQL 测试、E2E 预览
