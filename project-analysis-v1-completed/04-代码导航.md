## 6. RBAC 菜单 key 映射（易错）

`menu-items.tsx` 中 `MENU_PERMISSION_KEY` 与后端 `MENU_CATALOG`（`auth/permissions.ts`）**一一对应**，例如 `query` → `query`、`copilot` → `copilot`、`capability` → `capability`。

存量角色兼容：`expandLegacyMenuPermissions()` 在服务端自动扩展旧组合权限：

| 旧角色菜单 key | 自动追加 |
|----------------|----------|
| `connector` | `query` |
| `routing` | `capability` |
| `openapi` | `copilot` |

连接器 / ER 相关 API 使用 `@RequireAnyMenu('query', 'connector')`；查询测试、数据库连接工具 API 使用 `@RequireMenu('tool')`，需角色具备 `tool` 菜单权限。

**知识库 API 边界**：`/api/v1/knowledge-bases` 仅 CRUD 租户 wiki 绑定；内容/召回/向量化走 `/api/v1/knowledge/*`（wiki 代理）。自托管时代 stub 路由（data-sources/documents/retrieve/embedding-tasks）已下线。
