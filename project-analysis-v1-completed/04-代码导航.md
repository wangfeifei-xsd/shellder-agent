## 6. RBAC 菜单 key 映射（易错）

`menu-items.tsx` 中部分顶级 key 与后端 `@RequireMenu` 不一致：

| 侧栏 key | 实际权限 key |
|----------|--------------|
| `query` | `connector`（连接器页） |
| `copilot` | `openapi` |
| `capability` | `routing` |

查询测试、数据库连接工具 API 使用 `@RequireMenu('tool')`，需角色具备 `tool` 菜单权限。
