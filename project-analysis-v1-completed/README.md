## 给 AI / 开发者的使用方式

1. **改某个菜单功能**：先读 `modules/xx-*.md`，再按 `04-代码导航.md` 定位源码。
2. **改业务能力（问答/问数）**：读 `capabilities/` + `modules/03-查询型配置.md` 或 `modules/02-知识库.md`。
3. **改接入层（Copilot / OpenAPI）**：读 `modules/07-*.md`、`modules/08-*.md`。
4. **改安全 / 隔离相关**：先读 [01-范围与边界 §6](./01-范围与边界.md#6-已知缺口与后续强化) 与 [06-实施约束 §9–10](./06-实施约束-已落地.md#9-copilot--openapi-用户隔离目标态)。

**原则**：以 **当前代码与本目录文档** 为准；各模块文末「实现差异」仅作演进说明，勿覆盖现网行为。
