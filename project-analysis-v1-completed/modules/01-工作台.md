# 模块：工作台

| 属性 | 值 |
|------|-----|
| 菜单 | 工作台 / 首页 |
| 路由 | `/` |

---

## 功能

- Tool 调用统计：总数、成功/失败、成功率、失败率、平均耗时
- 高风险待确认列表（Top N）→ 链至 `/approvals`（**审批中心，已产品化**）
- 最近异常任务（Top N）→ 链至 `/tasks`
- 快捷入口（受 RBAC 菜单权限过滤）

---

## 代码

| 层 | 路径 |
|----|------|
| 前端 | `shellder-web-console/src/pages/console/page.tsx` |
| 后端 | `shellder-agent-server/src/dashboard/` |
| API | `GET /api/v1/dashboard/summary?tenantId=` |
| 权限 | `@RequireMenu('workbench')` |

---

## 数据来源

| 字段 | 表/模块 |
|------|---------|
| `toolStats` | `tool_call_audit` 聚合 |
| `pendingApprovals` | `approval`（审批中心） |
| `recentFailedTasks` | `task` status failed/timeout |

---

## 改造注意

- 工作台本身已验收；待确认/异常任务卡片在无对应模块权限时可能为空，属预期
