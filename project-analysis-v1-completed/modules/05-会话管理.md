# 模块：会话管理

| 属性 | 值 |
|------|-----|
| 初始方案 | 功能清单 §1.2 / 执行计划 08、16 |
| 菜单 key | `session` |

---

## 子菜单

| 子菜单 | 路由 |
|--------|------|
| 会话列表 | `/sessions` |
| 消息记录 | `/sessions/messages` |
| 调试台 | `/sessions/debug` |
| 会话详情 | `/sessions/:id`（无侧栏项） |

---

## 功能

| 页面 | 要点 |
|------|------|
| 列表 | 租户/用户/状态/能力类型/时间筛选 |
| 详情 | 消息流、工具节点、确认节点、上下文 |
| 消息记录 | 跨会话消息查询、类型筛选 |
| 调试台 | `POST /api/v1/sessions/debug` + 发消息 + **SSE** |

调试台走 **完整 Agent Runtime**，与 Copilot/OpenAPI 编排一致。

---

## 后端

| 模块 | API |
|------|-----|
| `session/` | `/api/v1/sessions` |
| `message/` | `/api/v1/sessions/:id/messages`、`POST /api/v1/messages` |
| `agent-runtime/` | `POST :id/messages`、`GET :id/stream`、确认相关 |

---

## 数据表

- `session`
- `message`

---

## 依赖说明

- 路由/技能书命中展示依赖 **能力路由、技能书**（实验中 UI）；Runtime 内仍可能调用
- 关联任务列表依赖 **任务中心**（实验中）

---

## 对照初始方案

§1.2 四项功能中，列表/详情/消息/调试台均已验收；调试台为 V1 核心验证入口之一。
