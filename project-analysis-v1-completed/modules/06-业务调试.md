# 模块：业务调试（能力演示）

| 属性 | 值 |
|------|-----|
| 菜单 | 业务调试 / 能力演示 |
| 路由 | `/capabilities` |

---

## 功能

- 管理后台内模拟 **嵌入式 Copilot** 完整链路
- 换票后调用 `/copilot/v1/*`，与 `/copilot` 嵌入页 **契约一致**
- 用于验收问答型、查询型、**操作型（含 HTTP 业务查询）** 等已配置能力

---

## 操作型 E2E 验收指引

1. **配置**：**连接器管理** → HTTP 连接器；**工具管理** → 查询工具 / 工具列表，完成 invoke 调试（可导出对接文档）
2. **本页**：选择 **操作型**，新建会话（写入 `capabilityType=action`）
3. **http_query**：自然语言（路由绑定 + LLM 抽参）或信号 `[查询工具:tool_code {...}]`；期望 `CapabilityResult.status=success`
4. **action**：输入与 Tool 描述匹配的指令；高风险可能返回待确认 `approvalId`
5. 详细说明见 [`capabilities/操作型能力.md`](../capabilities/操作型能力.md) §4
