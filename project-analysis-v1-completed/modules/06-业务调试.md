# 模块：业务调试（能力演示）

| 属性 | 值 |
|------|-----|
| 初始方案 | **无独立菜单**；调试验收分散在 §1.2 调试台、§2 Copilot |
| 菜单 | 业务调试 / 能力演示 |

## 功能

- 管理后台内模拟 **嵌入式 Copilot** 完整链路
- 换票后调用 `/copilot/v1/*`，与 `/copilot` 嵌入页 **契约一致**
- 用于验收问答型、查询型等已配置能力
