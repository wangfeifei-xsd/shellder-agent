# 模块：用户与权限

| 属性 | 值 |
|------|-----|
| 初始方案 | 功能清单 §1.10 / 执行计划 03 |
| 菜单 key | `user` |

---

## 子菜单

| 子菜单 | 路由 |
|--------|------|
| 用户管理 | `/users` |
| 角色管理 | `/roles` |
| 权限策略 | `/permissions` |

---

## 功能

| 模块 | 要点 |
|------|------|
| 用户 | 增删改、禁用、绑定租户、分配角色 |
| 角色 | 菜单权限（`menus`）、模块权限（`modules`）、Tool 范围（`toolScopes`） |
| 权限策略 | 四类能力访问、高风险审批权限（`policy` JSON） |

---

## 认证

| API | 说明 |
|-----|------|
| `POST /api/v1/auth/login` | 平台 JWT |
| `GET /api/v1/auth/me` | 当前用户 + 权限 |
| `GET /api/v1/auth/catalog` | 权限目录 |

**不与上层业务系统共享 Token**（实施规格 §1.5）。

---
