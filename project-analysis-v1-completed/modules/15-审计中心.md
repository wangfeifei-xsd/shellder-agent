# 模块：审计中心

| 属性 | 值 |
|------|-----|
| 菜单 key | `audit`（RBAC `audit.view`） |
| 状态 | **已产品化** |

---

## 信息架构

侧栏位于 **任务中心** 与用户与权限之间：

```
审计中心
├── 工具调用审计    /audit
├── 用户操作审计    /audit/user-actions
├── 外部接口审计    /audit/external-calls
└── 风险动作审计    /audit/risk-actions
```

---

## 功能

| 子页 | 功能 |
|------|------|
| 工具调用审计 | Tool 名称、调用人、时间、入参摘要、结果状态、耗时；按 Tool/租户/用户/状态筛选 |
| 用户操作审计 | 后台配置、审批、权限变更等操作记录；操作前后差异摘要 |
| 外部接口审计 | 外部系统接口调用；请求目标、结果状态、耗时、失败原因 |
| 风险动作审计 | 高风险动作申请→确认→执行→结果全链路；聚合审批记录与 Tool 调用审计，无第四套采集通道 |
