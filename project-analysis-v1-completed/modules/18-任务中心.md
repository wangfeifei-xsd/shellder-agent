| 子菜单 | 路由 |

---

## 功能

- 任务列表：按租户、状态、能力类型、时间筛选；支持同步/异步任务
- 任务详情：步骤（`task_step`）、输出、失败原因、耗时
- 长任务跟踪：流程型/异步型任务进度（`GET /api/v1/tasks/:id/progress`）
- 执行日志：任务级与步骤级日志（`GET /api/v1/tasks/:id/logs`）

### 与 Runtime / 工作台 / 流程编排联动

1. **流程型能力**执行时创建 `task` 与 `task_step`，步骤状态实时更新
2. `WorkflowToolInvoker` 按 `tool.config.workflow.steps` 顺序执行子 Tool；`previous_step` 绑定将前置步骤 output 注入后续 http_query / action 入参
3. 工作台「最近异常任务」卡片链至 `/tasks`
4. 会话详情可关联同 session 的任务列表
